Syskit: a Roby plugin for component-based systems

Syskit is a plugin for the Roby plan manager, that greatly simplifies the handling of complex component-based systems.

Syskit is currently developped within the Robot Construction Kit (http://rock-robotics.org). Have a look there, and in particular to the tutorials and the reference documentation
